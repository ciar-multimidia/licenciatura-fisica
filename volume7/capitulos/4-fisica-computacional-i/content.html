INFORMÁTICA BÁSICA

Ao ligar o computador, ele inicia o sistema operacional, o qual carrega uma série de programas com instruções para gerenciar os diversos recursos da máquina, entre eles podemos citar, o teclado, o mouse, leitor de CD/DVD, a memória RAM, os arquivos gravados no disco rígido da máquina (mais conhecido pela sigla inglesa HD de Hard Disk), etc. O sistema operacional é o responsável pelo gerenciamento dos dispositivos e além disso ele também faz uma interface entre os usuários e o hardware da máquina. Há diversos sistemas operacionais, sendo alguns proprietários e outros livres. Os mais conhecidos são o Windows, o qual é proprietário, e o GNU/Linux, o qual é livre. Há um número imenso de distribuições Linux baseadas no GNU/Linux, uma boa lista pode ser encontrada no sítio http://distrowatch.com.

Entre os sistemas operacionais livres mais conhecidos, podemos listar os seguintes:

  • GNU Operating System http://www.gnu.org/
  • Ubuntu Linux http://www.ubuntu.com/
  • Debian GNU/Linux http://www.debian.org/
  • openSUSE http://www.opensuse.org/pt-br/
  • FreeBSD http://www.freebsd.org/
  • FreeDOS http://www.freedos.org/
  • OpenSolaris http://opensolaris.com/ ou http://www.sun.com/software/solaris/freeware/
  • Chrome OS http://getchrome.eu

No sítio http://www.baixaki.com.br/tecnologia/2031-a-historia-dos-sistemas-operacionais.htm há um relato da evolução dos sistemas operacionais.

Software Livre

Como usuários de diversos softwares, devemos estar cientes que eles podem ser classificados quanto ao tipo de sua licença como:

  • Comercial.
  • Comercial liberados para testes.
  • Shareware (Liberado para testes por um período).
  • Freeware (Software gratuito).
  • Liberado para uso não comercial.
  • Binários de uso irrestrito.
  • Código aberto (Open Source Software) com licença do tipo:
    • BSD
    • Gnu Public License (GPL)
    • Artistic http://dev.perl.org/licenses/artistic.html
  • Copyleft:
    • Mozilla Public License (MPL)
    • Apache

As licenças citadas acima são as mais comuns, há inúmeras outras como as listadas no artigo "Várias Licenças e Comentários sobre elas" [1] da "Free Software Foutation" http://www.gnu.org/licenses/license-list.pt-br.html. Mas estamos mais interessados em esclarecer os conceitos de software livre e software gratuito. Basicamente um software gratuito (freeware), é todo aquele que você usar sem precisar pagar nada por ele.

Já o conceito de software livre é mais complexo, e segundo os critérios da http://www.gnu.org/home.pt-br.html Free Software Foundation (FSF) ou em português Fundação Para o Software Livre, é uma questão de liberdade e não de preço. De acordo com a FSF o termo Software livre refere-se à liberdade dos seus usuários poderem executar, copiar, distribuir, estudar, modificar e aperfeiçoar o software de acordo com suas necessidades, sem precisar de dar satisfação a ninguém.
